# mkt-writing-review — 마케팅 카피 AI 티 검토·윤문

작성된 제목·도입부·본문·CTA에서 AI 티(번역투·기계적 병렬·접속사 남발·이모지/불릿 과다 등)를
`/humanize-korean:humanize`로 검토·윤문한다. 내용(사실·수치·인용)은 불변, 문체만 자연스럽게.

## 언제
- mkt-blog-writing 에서 CTA까지 작성한 직후(발행 전 필수 검토).
- 단독: "AI틱 검토", "발행 전 카피 검토", "/mkt-writing-review".

## 연동
- 탐지·윤문 엔진: `humanize-korean` 스킬.
- 확정 후 mkt-blog-writing Step 6.5(썸네일)/Step 7(발행)로 복귀.
