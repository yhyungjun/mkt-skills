# Blog Output Template — 최종 출력 형식

Step 7(발행)에서 사용하는 최종 블로그 출력 형식이다.

---

## 출력 형식

### 메타 정보 블록

```
---
키워드: [선택된 키워드]
Limbic 유형: [traditionalist | performer | hedonist | disciplinarian | 없음(DAILY)]
예상 글자 수: [공백 포함 약 N자]
---
```

### 본문 형식

```markdown
# [제목]

[도입부]

[본문]

---

[CTA]
```

---

## 실제 출력 예시

```
---
키워드: 소프트웨어개발외주
Limbic 유형: traditionalist
예상 글자 수: 공백 포함 약 3,200자
---

# 소프트웨어개발외주, 믿을 수 있는 곳인지 확인하는 방법

[도입부 텍스트...]

[본문 텍스트...]

---

[CTA 텍스트...]
```

---

## Phase 2 발행 옵션

Phase 2(MCP)가 사용 가능한 경우, 발행 방법을 선택할 수 있다:

1. **복사용 텍스트** (기본): 위 형식으로 마크다운 출력 → 직접 복사/붙여넣기
2. **네이버 블로그 자동 발행**: `bonda:publish_naver(title, content)` MCP tool 호출
3. **인블로그 자동 발행**: `bonda:publish_inblog(title, content)` MCP tool 호출

발행 실패 시 자동으로 복사용 텍스트 출력으로 폴백.

---

## 플레이스홀더 치환

본문에 삽입된 아래 플레이스홀더는 발행 단계에서 실제 이미지 URL로 치환된다.

- `{{strength:UUID}}` — 강점(가치입증) 이미지 위치. UUID는 `company_strength` 테이블의 강점 ID.
- `{{image:이미지ID}}` — 커스텀 이미지 위치. 이미지 ID는 업로드된 커스텀 이미지 식별자.

---

## 출력 시 체크리스트

Step 7 출력 전 다음을 확인한다:

- [ ] 제목 글자수: 공백 제외 29자 미만
- [ ] 도입부: 공감 문단 + 가치입증 포함
- [ ] 본문: 소제목 2~4개, 해시태그 5~10개 포함
- [ ] CTA: 2~3문장, 공식 하나만 사용
- [ ] limbic_type 반영: 각 섹션에 심리 유형 반영 여부
- [ ] DAILY 유형: 키워드 "없음" 표시, CTA는 따뜻한 마무리
