# mkt-blog-writing — SEO 블로그 7단계 작성·발행 스킬

블로그 유형 선택 → Limbic 독자분석 → 키워드 → 제목 → 도입부 → 본문 → CTA → 썸네일 → 발행까지
**7단계**로 SEO 최적화 블로그를 작성·발행하는 [Claude Code](https://claude.com/claude-code) 스킬.

## 언제 발동
- "블로그 써줘", "SEO 블로그 작성", "블로그 글 만들어", "마케팅 블로그 글"
- "포트폴리오/도입사례 블로그", "/mkt-blog-writing"

## 워크플로우
1. 유형 선택(정보제공·자사홍보·포트폴리오·일상) + 정보 수집
2. Limbic 독자 분석(전통주의자·실행가·쾌락주의자·규율숭배자)
3. 키워드(고객여정 4단계 + 실검색량)
4. 제목(6공식) · 5. 도입부 · 6. 본문(PREP) · 7. CTA
8. (선택) 대표 썸네일 · 9. 발행

## 구조
```
SKILL.md            진입점(7단계 워크플로우·발동 조건)
reference/          단계별 참조(blog-types · limbic-types · customer-journey · prompts/*)
templates/          발행 출력 템플릿(blog-output.md)
```

## 연동 (선택)
- **bonda MCP**: 회사정보·키워드 조회·인블로그/네이버 발행·전환 추적링크
- **inblog CLI**: 초안 생성(슬러그·CTA·커버·태그), **발행 분리**(초안까지만)
- **썸네일**: 실사 배경형 V2(기본) / 4색 일러스트(요청 시)
- **CTA 버튼 카피**: 카테고리 규칙(`cta-config.md` SSOT)

## 주의
일부 경로·계정(조코딩AX 인블로그, `cta-config.md`, 실사 썸네일 템플릿)은 **작성자 환경 기준**이므로
자기 환경에 맞게 교체한다. MCP·CLI 미연동 시 각 단계는 복사용 마크다운 출력으로 폴백한다.
