# Limbic Types — 4가지 심리 유형 분석 가이드

Limbic 분석은 마케팅 심리학 기반으로 독자의 지배 감정 유형을 판단하는 것이다.
Step 1 완료 직후 자동 실행되며, DAILY 유형은 분석을 생략한다.

---

## 분석 방법

### INPUT

원본: `back/src/agents/common/prompts/limbic_analysis.py` — `LIMBIC_ANALYSIS_USER_TEMPLATE`

- `blog_type`: 블로그 유형 (INFO / PURCHASE / PORTFOLIO / DAILY)
- `purpose`: 글의 목적 (Step 1에서 추론)
- `extra_info` (유형별로 아래 필드를 조합하여 전달):
  - `target_audience`: 타겟 독자 (PURCHASE)
  - `target_problem`: 타겟이 겪는 문제 (PURCHASE/PORTFOLIO)
  - `target_desired_outcome`: 타겟의 원하는 변화/결과 (PURCHASE)
  - `user_provided_info`: 전달할 정보 (INFO)
  - `portfolio_customer_problem`: 고객이 겪던 문제 (PORTFOLIO)
  - `portfolio_solution`: 해결 방법 (PORTFOLIO)
  - `portfolio_result`: 고객의 변화 (PORTFOLIO)

### OUTPUT
```json
{
  "limbic_type": "traditionalist" | "performer" | "hedonist" | "disciplinarian",
  "dominant_emotion": "지배 감정 한 단어 (예: 불안, 지위 불만, 지루함, 비효율)",
  "reasoning": "판단 근거 1-2문장"
}
```

### 판단 규칙
- 글의 목적(purpose)에서 독자가 느끼는 핵심 감정을 추출한다.
- 타겟 문제(target_problem)에서 불안 / 지위 / 새로움 / 효율 중 무엇이 지배적인지 판단한다.
- **반드시 하나의 유형만 선택** — 혼합 불가.
- 판단 근거를 1~2문장으로 명시한다.

### DAILY / None 처리
- DAILY 유형: 분석 생략, `limbic_type = None`.
- `limbic_type = None` 이후 단계 처리:
  - 제목: 공식 제약 없이 자유롭게 작성
  - 본문: 문체 제약 없음, 자연스럽고 따뜻한 어조
  - CTA: 기본 따뜻한 마무리 ("앞으로도 함께해요", "공감되셨다면 좋아요~")

## 4가지 유형 상세

---

### 전통주의자 (traditionalist)

**지배 감정**: 불안, 안전 추구
**중시하는 것**: 검증된 것, 리스크 회피, 신뢰성, 안정성

**심리 특성**:
- 균형 시스템의 영향을 강하게 받는 유형이다.
- 새로운 것보다 검증된 것을 선호한다.
- 선택 시 안정성, 신뢰성, 품질, 검증된 사용 사례, 다른 사람들의 사용 여부를 중요하게 본다.
- 구매 결정 전에 세부 정보를 꼼꼼히 확인하고, 확신이 들지 않으면 조언을 구하거나 추가 정보를 찾는다.
- 위험이 낮은 선택을 선호하며 과도한 지출을 잠재적 위험으로 인식한다.
- 한 번 신뢰를 형성하면 단골 고객이 되는 경향이 강하다.

**제목 전략**:
- 신뢰, 안전, 검증을 강조하는 표현 사용
- 구체적인 사용 사례나 인증 언급
- "안전하다", "검증된", "위험 없이", "확인된" 류의 표현 활용

**본문 톤 및 문체**:
- 신뢰감 있고 차분한 어조
- 데이터, 사례, 인증으로 모든 주장을 뒷받침
- 단계별 설명으로 안심감 제공

**금지 표현**:
- "혁신적", "파격적", "전례 없는" — 불확실성을 높이는 표현 금지
- 과장, 근거 없는 주장 금지
- 감정적이거나 충동적 구매를 유도하는 표현 금지

**CTA 스타일**:
- 리스크 제거형: "부담 없이 상담받아보세요", "무료로 확인해보세요"
- 행동의 부담을 최소화하는 표현 사용
- "결정하기 전에 먼저 확인해보세요" 류의 저위험 CTA

---

### 실행가 (performer)

**지배 감정**: 지위, 경쟁 의식, 승리 욕구
**중시하는 것**: 성과, 우위, 속도, 프리미엄 이미지

**심리 특성**:
- 지배 시스템의 영향을 강하게 받는 유형이다.
- 핵심 감정은 승리, 인정, 지위 상승이다.
- 자신의 능력이나 위치를 타인에게 증명하는 것에 강한 동기를 가진다.
- 선택 시 경쟁 우위, 성과 향상, 빠른 결과, 지위 상승, 프리미엄 이미지를 중요하게 본다.
- 자신의 선택이 더 성공적으로 보이게 하는지 판단하며, 높은 비용도 기꺼이 지불한다.
- 감정적으로 충동적인 소비자가 아니라 목표와 성과를 중심으로 판단하는 전략적 소비자다.

**제목 전략**:
- 경쟁 우위, 성과, 속도를 강조
- "먼저", "앞서가는", "선도하는", "최고의" 류의 표현
- 구체적인 성과 수치 포함

**본문 톤 및 문체**:
- 전략적이고 행동 중심의 어조
- 구체적인 성과 수치 제시 ("40% 향상", "3배 빠른")
- 경쟁에서 우위를 점할 수 있다는 메시지 강조

**금지 표현**:
- "천천히", "여유롭게", "기다리세요" — 긴급성을 낮추는 표현 금지
- 느긋한 톤, 소극적 표현 금지

**CTA 스타일**:
- 우위 확보형: "지금 먼저 시작하세요", "경쟁사보다 앞서가세요"
- 행동이 곧 경쟁력이라는 메시지
- "이 기회를 놓치면 경쟁에서 뒤처집니다" 류의 긴박감 CTA

---

### 쾌락주의자 (hedonist)

**지배 감정**: 지루함 회피, 새로움 추구, 호기심
**중시하는 것**: 트렌드, 재미, 경험, 독특함, 감각적 디자인

**심리 특성**:
- 자극 시스템의 영향을 강하게 받는 유형이다.
- 핵심 감정은 지루함 회피와 새로움 추구다.
- 새로운 경험, 흥미로운 자극, 재미있는 활동에 강하게 끌린다.
- 선택 시 신제품, 트렌드, 독특한 경험, 감각적인 디자인, 재미있는 사용 경험에 반응한다.
- 얼리어답터 성향을 보이며 위험에 대한 경계심이 상대적으로 낮다.
- "재미있어 보이는가", "새로운 경험을 줄 수 있는가"가 가장 중요한 판단 기준이다.

**제목 전략**:
- 트렌드, 새로움, 신선함을 강조
- 궁금증과 호기심 자극
- "처음", "최신", "놀라운", "아무도 몰랐던" 류의 표현

**본문 톤 및 문체**:
- 트렌디하고 흥미로운 어조
- 최신 사례, 놀라운 결과, 의외의 발견 강조
- 생동감 있고 리듬감 있는 문장

**금지 표현**:
- 딱딱한 보고서 톤, 지루한 나열 금지
- 이미 알려진 뻔한 내용 금지
- 지나치게 진지하고 무거운 어조 금지

**CTA 스타일**:
- 호기심 자극형: "가장 먼저 경험해보세요", "놓치기 전에 확인하세요"
- 행동 자체가 재미있는 경험이 될 것이라는 느낌
- "이런 경험은 많지 않아요" 류의 희소 경험 강조

---

### 규율숭배자 (disciplinarian)

**지배 감정**: 비효율 거부, 통제 욕구
**중시하는 것**: 수치, ROI, 체계, 논리적 근거

**심리 특성**:
- 지배 시스템 중에서도 통제와 질서를 중시하는 유형이다.
- 세상을 예측 가능하고 통제 가능한 구조로 이해하려 한다.
- 선택 시 수치, 성능 데이터, ROI, 품질 보증, 객관적 평가를 중요하게 본다.
- 감정적 설득보다 논리적 근거와 데이터에 더 설득된다.
- 구매 전 다양한 정보를 비교 분석한다.
- 선택지가 너무 많으면 인지적 부담을 느끼므로 명확하게 정리된 선택 구조를 선호한다.
- 불필요한 기능이나 장식적 요소는 오히려 부정적으로 평가한다.

**제목 전략**:
- 수치, ROI, 효율을 강조
- "비용 절감", "시간 단축", "정량적" 류의 표현
- 비교 분석, 체계적 방법론 언급

**본문 톤 및 문체**:
- 논리적이고 구조적인 어조
- ROI, 비용 절감, 시간 단축 등 정량적 표현 중심
- 명확한 단계와 구조로 설명

**금지 표현**:
- 감성적/추상적 표현 금지 ("행복해진다", "만족스러운")
- 근거 없는 주장 금지
- 모호하거나 측정 불가능한 표현 금지

**CTA 스타일**:
- 분석 유도형: "비용 회수 시점을 계산해보세요", "효율을 직접 비교해보세요"
- 행동의 결과를 정량적으로 예측할 수 있다는 메시지
- "구체적인 수치로 확인해보세요" 류의 데이터 기반 CTA

## 유형 판단 빠른 참조

| 키 시그널 | 유형 |
|---|---|
| "안전한지 모르겠다", "리스크가 걱정", "검증된 곳이 좋다" | traditionalist |
| "경쟁사보다 앞서고 싶다", "빠르게 성과를 내야 한다", "남들보다 먼저" | performer |
| "새로운 시도가 좋다", "재미있어 보인다", "트렌드를 따르고 싶다" | hedonist |
| "ROI가 중요하다", "체계적으로 관리하고 싶다", "수치로 검증해야 한다" | disciplinarian |
